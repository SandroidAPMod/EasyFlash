# Flash Player Front-end “Easy Flash“

## Release / Download
https://github.com/SandroidAPMod/EasyFlash/releases/latest

## Introdução:

Esse é um Front-end que facilita a execução de games em flash, sem a necessidade de configurações  ou comandos.

## Como usar:

Descompacte o Arquivo: Easy Flash e Coloque seus games em formato .swf, na pasta games, depois execute Easy Flash.exe, Seus games apareceram na lista no centro da tela, escolha um game e clique em play.

## Possiveis Bugs:

Se o game não for executado de maneira correta, tente mudar o nucleo do flash, clicando na barra onde esta escrito: FlashPlayer32.exe
